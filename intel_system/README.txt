 설치 & 실행 (3단계)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[1단계] Gemini API 키 발급
  → https://aistudio.google.com 접속
  → 로그인 → "Get API Key" → "Create API key"
  → 키 복사

[2단계] config.py 수정
  → config.py 파일 열기
  → GEMINI_API_KEY = "여기에_발급받은_API_키_입력"
     발급받은 키로 교체

[3단계] 실행
  → 실행.bat 더블클릭
  → 브라우저에서 http://localhost:9999 자동 열림
